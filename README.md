# Nature Health & Wellness Home

This project is a simple website built from scratch with plain HTML, CSS, and JavaScript.
That makes it the best starting point for learning how websites work before moving to tools
like React, Next.js, or Shopify.

## Project files

- `index.html`: the structure and content of the page
- `styles.css`: the visual design, layout, colors, spacing, and responsive behavior
- `script.js`: small interactive features like the mobile menu and scroll animations

## Step-by-step: how this website was built

### 1. Start with the folder and files

Create a new project folder and add these files:

- `index.html`
- `styles.css`
- `script.js`

This is enough for a complete first website.

### 2. Build the page structure in HTML

The `index.html` file contains the page sections:

- `header`: the brand name and navigation
- `hero`: the main message visitors see first
- `about`: what the brand stands for
- `supplements`: the product categories
- `lifestyle`: educational wellness content
- `faq`: trust-building answers
- `contact`: the main call to action
- `footer`: brand and disclaimer

Think of HTML as the skeleton of the website.

### 3. Add the design in CSS

The `styles.css` file does four main jobs:

- defines the color palette with CSS variables in `:root`
- creates layout with Flexbox and CSS Grid
- styles buttons, cards, headings, and spacing
- makes the site responsive for tablets and phones

Important things to notice:

- `:root` stores reusable colors and sizes
- `.container` keeps content centered
- `.hero-grid`, `.feature-grid`, and `.product-grid` create multi-column layouts
- `@media` rules change the layout for smaller screens

Think of CSS as the visual system of the website.

### 4. Add small interactivity with JavaScript

The `script.js` file handles:

- updating the footer year automatically
- opening and closing the mobile menu
- animating sections into view while scrolling

Think of JavaScript as the behavior layer.

### 5. Keep your wellness messaging responsible

Because this site promotes supplements and natural wellness, your wording matters.

Good examples:

- "supports daily wellness"
- "helps build a healthy routine"
- "designed for balance and vitality"

Avoid strong medical claims unless you have proper legal and scientific support.
For example, do not casually say a product cures, treats, or prevents disease.

### 6. Open the website locally

You can view the site by opening `index.html` in a browser.

If you want a small local server, run:

```bash
python3 -m http.server
```

Then open `http://localhost:8000`.

### 7. What to change next

Once you understand this first version, improve it in this order:

1. Replace the sample text with your real brand story.
2. Add your real supplement names, ingredients, and product photos.
3. Create separate product pages.
4. Add a real contact form.
5. Add legal pages like privacy policy and terms.
6. Connect the site to hosting.

## A simple way to keep learning

Use this learning pattern every time:

1. Edit one section.
2. Refresh the browser.
3. Observe exactly what changed.
4. Repeat.

That is the fastest way to learn web development from scratch.
